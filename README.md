ORPAL AA-PI
==============


Antenna analyzer project for portable Linux-based devices. 
The current hardware implementation is based on BananaPi M2-Magic SBC (Allwinner R16).

Version 4.3

Based on the Yuri Kuchura's EU1KY_AA_V3 https://github.com/EU1KY/eu1ky_aa_v3


Build
==============

The build process depends on the platform you have chosen. Currently the AA-PI software is managed by repository https://github.com/orpaltech/armlinux.
